## 📲 How Different People Use It

### **Managers/Admins**
1. Open admin link on phone: `https://your-url.up.railway.app/login.html`
2. Login with admin credentials
3. Full access to:
   - Add/edit/delete products
   - Adjust stock with +/- buttons
   - View orders
   - Change admin password

### **Customers**
1. Open public link: `https://your-url.up.railway.app`
2. Browse products
3. Add to cart
4. No login needed

### **Team Members** (Same Admin Account)
1. Share the admin login link + password
2. Everyone accesses same dashboard
3. All changes show for everyone

## 🚀 Next: Deploy to Railway (5 Minutes)

1. Go to https://railway.app
2. Click "Start Now" → Connect GitHub
3. Select `brightenlighting` repo
4. Click "Deploy"
5. Wait for build (2-3 min)
6. Add environment variables (see DEPLOY_NOW.md)
7. **Done!** Share the links
